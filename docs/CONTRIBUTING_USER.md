# Contributing (User Guide)

Oxidized is a learning project focused on text editor architecture.
Contributions are welcome!

## Getting Started

1. Fork the repository and clone your fork
2. Install Rust 1.70+ and your preferred IDE
3. Build and test: `cargo build && cargo test`
4. Pick an issue from the tracker
5. Submit a PR with clear description

## Development Guidelines

- Code quality: follow Rust best practices; run `rustfmt`
- Testing: add tests for new behavior; keep suite green
- Documentation: update docs when behavior changes
- Architecture: maintain clean module boundaries and interfaces

## Areas for Contribution

- Implement roadmap features
- Performance and memory optimizations
- Cross‑platform improvements
- Documentation improvements
- Test coverage increases

For architecture-specific contribution notes, see
[CONTRIBUTING_ARCH.md](./CONTRIBUTING_ARCH.md).
